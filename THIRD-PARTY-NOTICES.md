## Additional Notes

### Attribution in Source Files

All bookmarklet source files in the `src/` directory contain header comments with:

- Original author attribution
- Source URL
- Copyright notice
- License information

### Build Process

The build process (`scripts/build.js`) preserves attribution information by:

1. Maintaining source file headers with copyright and license info
2. Adding console.log statements with source attribution in built output
3. Generating HTML pages with visible source links

### Compliance Commitment

We are committed to:

- Respecting all original licenses and copyrights
- Providing proper attribution
- Obtaining explicit permission where license terms are unclear
- Promptly removing any content upon request from copyright holders

### Contact

If you are a copyright holder and have concerns about the inclusion of your work, please open an issue at:
<https://github.com/pattespatte/public-bookmarklets/issues>
